## Apple Store Page: iPhone 13 Pro Max

### üìù Project Overview
This project is a simple **front-end interface** designed to simulate a product page for the **iPhone 13 Pro Max**. The main goal is to demonstrate how to use basic web technologies to create interactivity, specifically by changing the product image and the background color of the page based on user selection.

---

### ‚ú® Features

* **Color Swapping:** Users can click on the color icons at the bottom of the page to change the displayed iPhone image to a different color variant.
* **Dynamic Background Change:** The main container's background color (`.container`) automatically changes to complement the selected phone color, providing a cohesive visual experience.
* **Simple Layout:** The page includes a basic navigation menu, promotional text, and the main product image area.

### üõ†Ô∏è Technologies Used
This project was built using core web development technologies:

* **HTML5:** Used for structuring the page and organizing the content.
* **CSS3:** Used for styling, colors, layout (using Flexbox), and applying simple visual effects like the scale transition on icon hover.
* **JavaScript:** Used to add interactivity, specifically for dynamically swapping the phone images (`phones(phone)`) and changing the background color (`colors(color)`).
